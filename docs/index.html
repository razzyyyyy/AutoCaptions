Installation

1

Install Python

Download Python 3.8+ from python.org. Make sure to check "Add Python to PATH".

2

Download AutoCaptions

Download the latest release and extract the ZIP file anywhere on your computer.

3

Run the Installer

Double-click install.bat and wait for it to finish.

4

Use in DaVinci Resolve

Open your project, go to Workspace ‚Üí Scripts ‚Üí AutoCaptions
